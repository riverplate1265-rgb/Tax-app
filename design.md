# 手取り計算アプリ - UI デザイン計画

## アプリコンセプト
「素早く手取りが知りたい」をコンセプトにした簡易診断アプリ。
シンプルかつ信頼感のある金融アプリらしいデザイン。

## スクリーン一覧

### 1. ホーム画面（簡易診断入力）
- **目的**: ユーザーが情報を入力して診断を開始する
- **コンテンツ**:
  - アプリタイトル・サブタイトル
  - 生年月日入力（DatePicker or テキスト入力）
  - 年収入力（万円単位、数値キーボード）
  - 配偶者扶養あり/なし（トグルスイッチ）
  - 子供の人数（16歳以上19歳未満、19歳以上22歳以下）
  - 勤務地選択（都道府県ピッカー）
  - 「計算する」ボタン

### 2. 結果画面
- **目的**: 計算結果を分かりやすく表示する
- **コンテンツ**:
  - サマリーテキスト「年収◯◯万円のうち...」
  - 円グラフ（手取り・社会保険料・税金）
  - 内訳カード（社会保険料・所得税・住民税・手取り）
  - 注意書き（標準報酬月額・前年所得ベース）
  - 「もう一度計算する」ボタン

---

## キーユーザーフロー

1. ホーム画面で入力 → 「計算する」タップ
2. 結果画面にスライドアニメーションで遷移
3. 円グラフ・内訳を確認
4. 「もう一度計算する」でホームに戻る

---

## レイアウト方針

- **ナビゲーション**: シングルスタック（タブなし）、ホーム → 結果の2画面構成
- **フォーム**: ScrollView でスクロール対応
- **入力UI**: iOS ネイティブ風のセルスタイル（グループ化リスト）
- **グラフ**: react-native-svg を使用したドーナツチャート
- **フォント**: システムフォント（SF Pro / Roboto）
